## ロードマップ

### Phase 0: 境界面シミュレーション（現在）

2D multi-channel Lenia を「境界面」として実装し、
単純なルールから複雑な構造が創発するパラメータ空間を探索する。

- [x] wgpu + winit でリアルタイム描画エンジン構築
- [x] 3フィールド Lenia シミュレーション実装
- [x] Laws アーキテクチャ（9メタパラメータ → 全ランタイムパラメータ導出）
- [x] パラメータ探索ツール（mutation, fine-tuning, per-field調整, プリセット）
- [x] 3つの物質が非平衡で共存するパラメータ領域の発見（F1プリセット）
- [ ] ソリトン（自己制限的安定構造）の全フィールド同時達成
- [ ] 飽和後の動的変化の実現

### Phase 1: 物質と性質

境界面上の安定パターンを「物質」として認識・分類する。

- 安定構造の自動検出
- パターンへの性質付与（密度、移動性、反応性など）
- 構造間の相互作用ルール

### Phase 2: 3Dへの射影（ホログラフィック原理）

2D境界面の情報を3Dゲーム世界に射影する。

- 2D パターン → 3D オブジェクトへのマッピング
- 境界面の状態変化が3D世界に反映される仕組み

### Phase 3: ゲームプレイ

- プレイヤーの操作と世界との相互作用
- クラフト（性質ベースの合成、レシピではない）
- 創発的な連鎖反応とプレイヤーによる発見
