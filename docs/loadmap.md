## 未解決の設計課題

### 階層構造の設計

3つのソリトン（R,G,B）は素粒子に相当する基底層であり、
プレイヤーに直接見せる対象ではない。

```
ソリトン（素粒子） → 見えない基底層。3つの場の安定構造。
  ↓ 相互作用
原子的構造         → 複数ソリトンの束縛状態。ここから上がゲーム世界。
  ↓ 結合
分子的構造         → プレイヤーが触れる「物質」
  ↓ 組織化
生物的構造         → 創発的な複雑系
```

核心の設計課題は **3つの素粒子ソリトンの相互作用から、
どうやって次の階層（原子）が自然に創発するか**。

現在の cross_weight の反対称結合は素粒子レベルの力に相当する。
ここから「原子」が生まれるには、複数のソリトンが束縛状態を形成する必要がある
（例：R,G,B のソリトンが互いの周りを軌道的に回る、
空間的に重なって複合構造を作る、など）。

### 真のソリトン vs 準安定構造

F1プリセットの構造は安定して存在し続けるが、厳密にはゆっくり膨張する（準安定）。

- **真のソリトン** — 9パラメータだけでサイズ・形が完全に決まる。
  素粒子としての性質が環境に依存しない。上位階層の創発に必要な安定基盤。
- **準安定構造** — 資源制限や空間制約で成長を抑える設計も可能。
  成長・侵食自体が上位階層のダイナミクスになりうる。

### 同一パラメータ内の相のバリエーション

同じパラメータセットから初期条件の違いにより複数の安定パターンが生まれる
（結晶的対称構造、アモルファス迷路構造など）。
素粒子レベルでは単一の振る舞いが望ましいか、
相のバリエーション自体が上位階層の多様性の源泉になるかは未定。
